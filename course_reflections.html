Course Reflections

This course introduced me to the foundational principles of algorithm design, optimization, and problem-solving techniques. It helped me understand how to tackle complex computational challenges effectively and efficiently.

1. Binary Search Tree (BST)

A binary search tree is a hierarchical data structure where nodes are arranged so that the left subtree contains smaller elements and the right subtree contains larger elements. Visualizing tree structures and their dynamic updates during insertion and deletion was challenging. However, understanding how BSTs are used in databases for efficient data retrieval was an exciting application.

2. DFS and BFS (Graph Traversals)

DFS explores nodes as deep as possible before backtracking, while BFS explores nodes level by level. I found it important to understand when to use BFS for shortest path problems and DFS for backtracking problems.

3. Heap

A heap maintains the heap property, where the root is the minimum (min-heap) or maximum (max-heap) element. Implementing heaps for priority queues and real-time scheduling applications was a crucial learning experience.

4. Sorting Algorithms

Sorting algorithms like quicksort, mergesort, and bubble sort were studied. I learned to select the most appropriate sorting algorithm based on dataset size and constraints.

5. Pattern Searching

Algorithms like KMP and Rabin-Karp efficiently search for substrings in larger strings. I appreciated the optimization and pattern-matching techniques employed in large-scale search engines.

6. Graph Algorithms

I worked on Dijkstra's (shortest path), Kruskal's (MST), and other graph algorithms, learning their applications in real-world systems like GPS navigation and social networks.
